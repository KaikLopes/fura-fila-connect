# 📅 agenda.html (Página)
Interface principal de operação da clínica.

## 🧱 Componentes de UI
- **`WeekPicker`**: Seletor de data horizontal.
- **Tabela de Atendimentos**: Listagem central com ações rápidas.
- **Barra Lateral de Fila**: Exibição compacta de pacientes em espera para o dia.

## 🪟 Modais Gerenciados
- **`slotModal`**: Formulário de novos agendamentos com busca dinâmica de pacientes.
- **`checkoutModal`**: Painel de finalização financeira.
- **`waModal`**: Interface de chat para comunicação direta via WhatsApp.
- **`detailsModal`**: Visualização expandida de dados do atendimento.
